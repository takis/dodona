Schrijf een functie neg_pos_nul die een parameter x
neemt, en als resultaat "negatief", "nul" of "positief"
op het scherm toont.

### Input

Een getal.

### Output

Negatief, nul of positief.

### Example

**Input:**

      124

**Output:**

      positief

### Example

**Input:**

      -99

**Output:**

      negatief

### Example

**Input:**

      0

**Output:**

      nul
