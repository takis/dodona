Schrijf een functie `dict_increment()` die 1 optelt bij iedere waarde in een dictionary.

### Input

Een dictionary met als waarden integers.

### Output

De aangepaste dictionary met de waarden met 1 opgehoogd.

### Example

**Input:**

      {"A": 10, "B": 20}

**Output:**

      {"A": 11, "B": 21}
