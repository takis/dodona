Schrijf een functie lijstletters die een parameter x heeft
en als resultaat een lijst met twee letters geeft:
de eerste letter en de laatste letter.

### Input

Een string.

### Output

Een lijst met de eerste en laatste letter van de string.

### Example

**Input:**

      "abc"

**Output:**

      ["a", "c"]

### Example

**Input:**

      "Hallo"

**Output:**

      ["H", "o"]
