Schrijf een functie datumstukjes die een parameter x heeft
in formaat jaar/maand/dag 
en als resultaat meerdere retourwaardes geeft:
de dag, de maand en het jaar

### Input

Een string in formaat yyy/mm/dd.

### Output

dag, maand, jaar

### Example

**Input:**

      2021/01/11

**Output:**

      11,01,2021

### Example

**Input:**

      1976/12/20

**Output:**

      20,12,1976
