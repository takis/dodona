Schrijf een functie bmi die parameters gewicht en lengte heeft
en als resultaat het BMI geeft.

### Input

Een getal gewicht en een getal lengte

### Output

BMI

### Example

**Input:**

      75, 1.5

**Output:**

      32.0

**Input:**
